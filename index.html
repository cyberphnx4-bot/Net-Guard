Add DNS Rule
Create rules to control access to specific domains. You can block ads, allow specific sites, or redirect traffic to a different IP address.
Domain / Pattern
Enter the domain name (e.g., example.com). Use * as a wildcard for subdomains (e.g., *.ads.example.com matches all subdomains of ads.example.com).
Action
Block: Prevents access (returns 0.0.0.0)
Allow: Permits access (bypasses other rules)
Redirect: Sends traffic to a specific IP
Redirect IP
Enter the local or public IP address where you want to send the traffic. Useful for setting up local servers or captive portals.
Router Scope
All Routers: Rule applies to the entire network
Specific: Select multiple routers
Single: Apply to just one router
Active Rules
Manage your existing DNS rules here. You can edit, delete, or temporarily disable rules. Use the filters to find specific rules.
Filters
Narrow down the list by Action (Block/Allow) or Router. Use the search bar to find rules by domain name.
